A 3D vector. Has built in functions for vector operations (addition, subtraction, multiplication, etc).

### Fields
| Field | Type | Description |
| :---- | :--: | ----------- |
| `X` | `number` | The X component of the vector. |
| `Y` | `number` | The Y component of the vector. |
| `Z` | `number` | The Z component of the vector. |

### Functions
| Function | Description |
| :----- | ----------- |
| [`Length`](#vector3length) | Calculates the length of a vector. |
| [`Dot`](#vector3dot) | Returns the dot product of this vector and another. |
| [`Normalize`](#vector3normalize) | Normalizes the vector. |
| [`Cross`](#vector3cross) | Returns the cross product of this vector and another. |
