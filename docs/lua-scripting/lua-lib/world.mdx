The world class provides access to many utility functions used to access categories and pathables (markers/trails) within the world.

### Functions
| Function | Description |
| :----- | ----------- |
| [`CategoryByType`](#worldcategorybytype) | Finds and returns a category based on the provided type. |
| [`PathableByGuid`](#worldpathablebyguid) | Returns a pathable based on the provided guid. |
| [`PathablesByGuid`](#worldpathablesbyguid) | Returns a table of pathables based on the provided guid. |
| [`MarkerByGuid`](#worldmarkerbyguid) | Returns a marker based on the provided guid. |
| [`GetClosestMarker`](#worldgetclosestmarker) | Returns the closest marker. |
| [`GetClosestMarker`](#worldgetclosestmarkercategory) | Returns the closest marker directly within the specified category. |
| [`GetClosestMarkers`](#worldgetclosestmarkers) | Returns the specified number of the closest markers to the player. |
| [`GetClosestMarkers`](#worldgetclosestmarkerscategory-quantity) | Returns the specified number of the closest markers to the player within the specified category. |
| [`TrailByGuid`](#worldtrailbyguid) | Returns a trail based on the provided guid. |

### <CodeHeader class="World" function="CategoryByType" params="type">World\:CategoryByType</CodeHeader>
Finds and returns a category based on the provided type.

#### Parameters
| Parameter | Type | Description |
| :-------- | :--: | ----------- |
| `type` | `string` | The type of category to find. |

#### Returns | [`category`](category)
The category found, or nil if not found.

#### Usage
```lua
local category = World:CategoryByType("example.category.type")
```

### <CodeHeader class="World" function="GetClosestMarker" params="category">World\:GetClosestMarker(category)</CodeHeader>
Returns the closest marker directly within the specified category.

#### Parameters
| Parameter | Type | Description |
| :-------- | :--: | ----------- |
| `category` | [`Category`](category) | The category to search within. |

#### Returns | [`Marker`](marker)
